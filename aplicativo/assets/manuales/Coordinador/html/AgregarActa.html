Agregar Acta


En la pantalla principal de seguimiento, hay una opcion que permite agregar las actas de las actividades.


Creación de acta


Al seleccionar la opción "Agregar Acta" se crea un documento con unos estilos preestablecidos, el cual facilita la creación de las actas de cada actividad, éste documento presenta campos como:

- Participación
- Objetivos
- Acuerdos


Al igual que los productos, las actas posee un campos de palabras claves y un campo que permite seleccionar si el documento es publico o no.
