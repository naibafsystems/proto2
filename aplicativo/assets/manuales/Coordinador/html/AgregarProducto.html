Agregar Producto


En la pantalla principal de seguimiento, hay una opcion que permite agregar los productos o resultados que producen las actividades.


Formulario de producto


Al seleccionar la opción de "Agregar Producto" el sistema cargara un formulario con dos campos de texto, un campo de selección y un campo para subir archivo.

Los campos de texto deben completarse con las observaciones del producto y algunas palabras clave para su identificación, por otro lado, el campo de selección permite elegir si el documento es publico o privado.
