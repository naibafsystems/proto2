Actividades


En la pantalla principal de grupos de trabajo, se cuenta con la opción para acceder a las actividades que pertenecen al grupo de trabajo.


Pantalla principal


En la pantalla principal de actividades, encontrara una tabla con toda la información de las actividades que posee el grupo de trabajo. También esta incluido un botón que permite acceder a la creación de nuevas actividades para el grupo.


Buscar


Para desplazarse de forma mas fácil por la lista, encontrara una opción que le permitirá buscar una actividad de forma especifica, para realizar el filtro, el coordinador puede utilizar la descripción de la actividad.


Exportar lista


La aplicación cuenta con una opción la cual le permite exportar el listado de las actividades pendientes a la fecha, el coordinador podrá escoger el formato en el cual desea exportar el listado(Microsoft Excel o PDF).
